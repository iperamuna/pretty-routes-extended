# Release Notes - v1.0.0

Interactive and beautiful route visualization for Laravel, powered by Livewire and Sushi.

## ‚ú® Features
- **Premium UI**: Modern Tailwind CSS design with high-quality aesthetics.
- **Real-time Search**: Instant search by URI, name, or action.
- **Hierarchical Filtering**: Group and filter routes by prefix.
- **Interactive Highlighting**: Toggle row highlights for easier tracking of specific routes.
- **Mirrored Scroll**: Sticky footer with a synced horizontal scrollbar for long URIs.
- **Analytics**: Live route count statistics that respect current filters.
- **Configurable**: Easily control access, endpoints, and branding via configuration.
